# Mega RenewHub

A lightweight offline desktop application for managing digital asset renewals, including domains, hosting accounts, and software licenses. Mega RenewHub helps users track expiry dates, receive smart alerts, and keep all asset details organized—without needing an internet connection.

## About This Release

This repository provides the **compiled Windows executable (.exe)** of Mega RenewHub for users who want to run the software without installing Python or viewing the source code.

**No source code is included in this public repository.**

---

## Features

* 100% offline desktop app
* Domain, hosting, and software license tracking
* Smart expiry alerts (30, 10, and 1 day before expiry)
* Clean and organized UI with sidebar navigation
* Local SQLite storage for fast performance
* Developed with Python & Tkinter

---

## How to Use

1. Download the `.exe` file from the Releases section
2. Extract the ZIP (if packaged)
3. Run `MegaRenewHub.exe`
4. Start managing your renewals instantly—no installation required

---

## System Requirements

* Windows 7/10/11
* 4GB RAM or above
* No internet connection required
